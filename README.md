🎓 报告评分系统 - 操作指南 (Grading System Guide)
本模块用于处理 SOVO 项目期末报告。它采用“人机协同”模式：程序负责提取文档内容和合成最终成绩，大模型（如 DeepSeek/Kimi）负责理解内容并打分。

1. 准备工作 (Preparation)
在运行前，请确保您手头有以下两类文件：

学生报告文件：

格式：支持 .docx (Word) 或 .pdf。

命名建议：第X组_项目名.docx（文件名将作为小组的唯一标识）。

学生花名册 (Roster)：

格式：.xlsx (Excel)。

⚠️ 必须包含的列名： 表头中必须严格包含 姓名 和 学号 这两列。程序依靠学号进行精准匹配。

2. 操作流程 (Step-by-Step)
这个工具分为三个阶段，请按顺序操作：

阶段一：文档解析 (Tab 1)

上传： 在“📁 文档处理”标签页，批量上传所有小组的报告文件。

处理： 点击 “🚀 开始处理文件”。程序会提取所有文本和图片数量。

生成指令： 处理完成后，下方会自动生成一段超长的 AI 提示词 (Prompt)。

动作： 点击 “📥 下载指令文件” 或直接复制那段 prompt。

阶段二：AI 打分 (Human-in-the-Loop)

发送给 AI： 打开您常用的 AI 工具（推荐 Kimi、DeepSeek 或 ChatGPT）。

粘贴指令： 把刚才下载的 Prompt 发给 AI。

获取结果： AI 会按照指令要求，返回一个 CSV 格式的代码块（包含文件名、分数、组员名单、权重）。

动作： 复制 AI 返回的这段 CSV 文本。

阶段三：成绩合成 (Tab 2 & 3)

导入 AI 数据： 回到工具的 “🧮 成绩计算” 标签页，将 CSV 文本粘贴到 “AI 评分结果” 输入框中。

导入花名册： 上传您的《学生花名册.xlsx》。

上帝模式（可选）： 如果某些小组有额外加分（如路演表现优秀），在“特权加分配置面板”中手动调整。

计算与导出：

点击 “🚀 计算最终成绩”。

前往 “📈 数据分析” 标签页，下载最终的 《成绩单.xlsx》。

3. 常见问题 (Troubleshooting)
报错：KeyError: '学号' 或 KeyError: '姓名'

原因： 您的花名册 Excel 表头不对。

解决： 请打开 Excel，将对应的列名修改为标准的“学号”和“姓名”。

报错：缺少列: 文件名, AI基础分...

原因： 粘贴给 AI 的 Prompt 没执行好，或者您粘贴回来的 CSV 格式缺损。

解决： 检查 AI 的回复，确保它输出的是标准的 CSV 格式。如果 AI 废话太多，请强调用 Prompt 里的“只输出 CSV 代码块”要求它重写。

乱码或提取失败

原因： 某些 PDF 是纯图片扫描版，无法提取文字。

解决： 建议使用 .docx 源文件，或者使用带 OCR 功能的 PDF 工具预处理。
